# pyorbits
Fast and light python package simulating 2D orbits using JIT compilation (Numba). The project is implemented under RSE Python course on AGH Univeristy.

## Documentation

Full API documentation is available here:
[https://i-hejmej.github.io/orbit-euler/](https://i-hejmej.github.io/orbit-euler/)

## How to use

To install the package, this will do:
```bash
!pip install --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple/ pyorbits 
```
An example of usage is contained in [demo_pyorbits.ipynb](demo_pyorbits.ipynb).
